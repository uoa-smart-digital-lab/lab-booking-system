Meta Quest 2

Description
This is a Virtual Reality device developed by Facebook, now called Meta.
Information
We have two types of Meta Quest 2 - The business version without the AppStore (eg QUEST2_B_1) and the consumer version (eg QUEST2_C_1).
Both have the same capabilities, but presently, only the consumer version can be used in 'link' mode.
Find out more on the Meta Website.
Programming
There are numerous guides on the Smart Digital Lab canvas page to help you get going with programming in Unity.
